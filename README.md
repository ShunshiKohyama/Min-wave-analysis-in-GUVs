# Min-wave-analysis-in-GUVs
ImageJ macro for the detection of GUVs, making kymographs of Min waves inside the GUVs, and estimation of the period of waves from LSM or relevant image files.

**Min_wave_analysis_in_GUVs.ijm**
Step-by-step detection, processing (including drift fixing etc.), make Kymographs (can be saved), and extract Min wave chracteristics (wave mode, oscillation periods etc).

**Min_wave_fast_profile_JS.js**
Fast js (hybird with ijm) script to detect and extract Min wave profile (equivalent to the kymograph) in CSV formats for the down stream analysis.

**Min_wave_fast_profile_JS_Batch.js**
Batch process mode of profile extraction. It search the all .tiff files in the specified directory and output the CSV files in the same directory.
